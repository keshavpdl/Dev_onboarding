# Keela CRM
This is Simple CRM Based On Keela Platform

## How to use
1. Clone this repo
2. Run meteor npm install
3. Run meteor
4. Open http://localhost:3000 in your browser
5. Login with your credentials

- [Meteor](https://www.meteor.com/)
